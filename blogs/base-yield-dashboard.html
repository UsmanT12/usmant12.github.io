What It Does

A comprehensive portfolio dashboard specifically designed for the Base network. The app allows users to connect their wallet and view:

  • Token balances (ERC-20 tokens like ETH, USDC, AERO) with real-time values
  • Lending and borrowing positions across DeFi protocols
  • Health factors (risk indicators for liquidation)
  • Yield earnings from various DeFi positions
  • Unified interface consolidating multiple protocol interactions

The dashboard is optimized for Base Layer 2, providing responsive design for both desktop and mobile devices.

Key Features

  • Wallet Integration: Connect via MetaMask or WalletConnect for seamless access
  • Multi-Token Support: View balances for all ERC-20 tokens in your wallet
  • Protocol Integration: Direct integration with lending protocols like Seamless
  • Risk Monitoring: Real-time health factor tracking to prevent liquidation
  • Dual Network: Supports both Base Mainnet and Sepolia Testnet
  • Direct On-Chain Reads: No intermediary APIs for core functionality

Architecture & Data Flow

1. Wallet Connection

User connects wallet using RainbowKit, which supports MetaMask, WalletConnect, Coinbase Wallet, and more. The connection state is managed by wagmi hooks.

2. Balance Fetching

Once connected, the app reads token balances directly from smart contracts using ethers.js:

  • Fetches native ETH balance from the network
  • Queries ERC-20 contracts for token balances
  • Aggregates all balances for display

3. Protocol Position Reading

For integrated protocols (like Seamless), the app:

  • Calls lending pool contracts to get supply positions
  • Fetches borrow positions and outstanding debt
  • Calculates health factors using protocol formulas
  • Retrieves current APY rates for both supply and borrow

4. Real-time Updates

The dashboard uses React hooks to poll for updates periodically, ensuring users see current balances and positions. Critical metrics like health factors update more frequently.

Health factor is calculated based on collateral value vs. borrowed value:

DeFi Concepts Explained

Lending & Borrowing

Supply assets to earn yield, or borrow assets by providing collateral. Interest rates are typically determined algorithmically based on supply and demand.

Health Factor

Ratio indicating how safe your borrowed position is. Higher is safer. If it drops below 1.0, your collateral may be liquidated to repay the loan.

APY (Annual Percentage Yield)

The yearly rate of return, including compound interest. Used to compare different yield-earning opportunities.
